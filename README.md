# Superstore Sales Analysis Dashboard (Excel)

## Project Overview
This project analyzes the Superstore retail dataset using Excel to identify sales trends, product performance, and profitability patterns.

The goal was to organize raw sales data and transform it into a clear and interactive dashboard for business insights.

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data formatting and filtering

## Dashboard Highlights
The Excel dashboard focuses on key sales and profitability metrics:

- Sales and profit performance across regions
- Category-wise sales distribution
- Monthly sales trends
- Discount impact on profit
- Top performing products and sub-categories

## Key Insights
- Technology products show strong sales performance.
- Some products generate significantly higher profits than others.
- Discounts can negatively affect profit margins.
- Sales trends vary across time and regions.

## Skills Demonstrated
- Data analysis using Pivot Tables
- Dashboard creation in Excel
- Data organization and formatting
- Identifying business insights from retail datasets
